Book Record System (C++)
Description

This project is a console-based Book Record System developed in C++ as part of my programming practice and laboratory exercises. The program demonstrates the fundamentals of file handling, exception handling, arrays, and user input validation while managing a simple collection of book records.

The system allows users to enter information for three books, including the Book ID, Title, Author, Price, and Quantity. Each record is validated using C++ exception handling to ensure that empty fields, invalid prices, and negative quantities are detected before saving.

Book records are stored in a text file (books.txt) using file handling and can later be loaded and displayed. This project also includes searching for books by title and sorting records alphabetically, providing hands-on experience with array manipulation and basic algorithms.

Features
 Add book records
 Search books by title
 Display all saved books
 Sort books alphabetically by title
 Save records to a text file
 Load records from a text file
 Basic exception handling for invalid inputs
 File error checking when opening files
